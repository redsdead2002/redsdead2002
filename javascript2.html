Events

Events are actions that happen in the system you are using, The system fires a signal for each event, so we can do something when it happens.



			//Html code
					<button id="demo">Click Me</button>

			// javascript code
					let btn = document.getElementById("demo");

			// This it the event handler
					btn.onclick = function(){
					alert("The button was clicked.");
					};

The switch Statement

if your code contains a lot of if statements then the switch Statement can be used to make your code easier to read.
The first example shows the color of the given car by checking what car is specified.


								
			let car = "ford";
						
			if(car == "ford") {
				document.getElementById("demo").innerHTML = "The color is red"; // This code is run
			}
			if(car == "renult") {
				document.getElementById("demo").innerHTML = "The color is yellow";
			}
			if(car == "mazda") {
				document.getElementById("demo").innerHTML = "The color is blue";
			}

The second example uses the switch statement instead of the if statement.


												
			let car = "ford";
										
			switch(car) {
				case "renult": 
					document.getElementById("demo").innerHTML = "The color is red";
					break;
					case "ford": 
					document.getElementById("demo").innerHTML = "The color is yellow"; // This code is run
					break;
					case "mazda":
					document.getElementById("demo").innerHTML = "The color is blue";
					break;
			}

As you can see the code is easier to read.

The switch statement matches an expression against given values in the case clauses.
The switch statement evaluates the expression to value1, then value2 and so on, If the experssion matches the given value, Then the associated code block is run.
syntax:

													
							switch(expression) {
								case "Value1": 
									// Code block to run
									break;
								case "Value2": 
									// code block to run
									break;
								case "value3":
									// code block to run
									break;
							}

The break keyword

Notice the break keyword in the examples above, its roll is to stop the code execution after the evaluation matches a case. In the example below with no break keyword all the code runs.


																	
			let car = "ford";
															
				switch(car) {
					case "ford": //car is ford so this code will run
						document.getElementById("demo").innerHTML = "The color is red";
						// no break
					case "renult":   // every following case will now run
						document.getElementById("demo").innerHTML += "The color is yellow"; // This code is run
					case "mazda":
						document.getElementById("demo").innerHTML += "The color is blue";
							}

The default Clause

If the expression does not match a case the default clause is executed.


																						
			let car = "sibb";
																				
			switch(car) {
				case "ford": 
					document.getElementById("demo").innerHTML = "The color is red";
					break;
				case "renult":  
					document.getElementById("demo").innerHTML += "The color is yellow"; 
					break;
				case "mazda":
					document.getElementById("demo").innerHTML += "The color is blue";
					break;
				default:
					document.getElementById("demo").innerHTML = "That is not a car";
					}

A for loop repeatedly executes statements as long as the condition is true,
Syntax:

			
					for(initialExpression; condition; incrementExpression) {
						//statements or code blocks
					}

  • initialExpression -to initializes the loop counter.
  • condition - If true, the loop will run, if false code will not execute.
  • incrementExpressionset the increment of loop counter. Runs everytime the statement runs.

The statement will run 5 times.

			
					for(let i = 0; i < 5; i++) {
						document.write("<p>Hello World!</p>")
					}

The loop counter is initialized with the value of i = 0, them the condition is set, i must be less than 5 for the loop to run, The loop is then incremented, i++ increments the counter everytime the code block is executed.

For...of loop

The for...of statement is used to loop over iterable objects, which includes arrays and strings.

			
					for(variable of iterable) {
						//statements or code blocks
					}
  • variable -At each iteration the value is assigned to variable.
  • iterable -The object being iterated over

Below you can see it loop over an array.

	
			let names = ["Dave", "Steve", "scott", "paul"];		
			for(value of names) {
				document.write(value + "<br>");
			}

This example shows a loop over a string.

	
			let text = "Dave";	
			for(letter of text) {
				document.write(letter + "<br>");
			}

The while loop

The while loop will execute statements as long as conditions are true.
Syntax:

	
			while(condition) {

			}
  • condition -The expression evaluated, if true code will run, if false will not run.
  • statements -code block to run if conditions are true.
