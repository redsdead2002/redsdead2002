Operator Sequence

The operator sequence sets the order of performed operations in a math's expression.


	let num = 10 + 3 * 5;    // Division performed first so 3 * 5 = 15, 10 + 15 = 25
	document.write(num);

You may have noticed that the result of the expression above is 25 and not 65 as expected, This is because multiplication is performed before addition.

3 * 5 = 15 and then 10 + 15 = 25

The operator sequence is multiplication is performed first then division second, addition is third and subtraction last. If you want to perform say addition first then we can use grouping for that, grouped expressions are performed first before the others, just use parentheses () to group expressions.

let y = (10 + 3 ) * 5;.

The above expression returns 65 because 10 + 3 = 13, then 13 * 5 is 65.

forEach() Method

The forEach() method executes the given function once for each array element, and has only one argument a callback function.
The callback function will accept one of three arguments:

  • value- The element being processed.
  • index- The index of the current elements.
  • array- The array itself.
The index and the array elements can be omitted.


		let car = ["ford", "audi", "saab", "Mazda"];
				
		car.forEach(function(value, index, array){
		document.getElementById("demo").innerHTML += '<p> Index: ' + index + ' <br>Value: ' + value + '</br>';
		});

map() Method

The map() Method creates a new array with each element changed according to the logic provided by the function.
The given function will accept one of three arguments:

  • value- The element being processed.
  • index- The index of the current elements.
  • array- The array itself.


		let numbers = [10, 20, 30, 40, 50];

		let numbersTwoTimes = numbers.map(function(value) {
		return value * 2;
		});

		document.getElementById("demo").innerHTML += numbersTwoTimes;

filter() Method

The filter() Method creates a new array with each element that passed the given test by the provided function.
The given function will accept one of three arguments:

  • value- The element being processed.
  • index- The index of the current elements.
  • array- The array itself.
Only the numbers less than 40 are passed to the new array.


		let numbers = [10, 20, 30, 40, 50];

		let newArr = numbers.filter(function(value) {
		return value < 40;
		});

		document.getElementById("demo").innerHTML += newArr;
